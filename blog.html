Displaying interactive phylogenies

A list of software capable of being embedded into html, Feb 18, 2020

In recent years (and months) there has been much development on the display of phylogenies in browsers using javascript. Jason Davies created the very useful parseNewick.js function (see here) a few years back, which allows for the conversion of the Newick file format into a .json object. Manual creation of a tree is then possible, though difficult (see here).

There are several packages which offer full tree plotting and manipulation. The first I have come across is Phylotree.js which also offers an interactive web-app where you can upload your own trees and mess around with the multiple options. The only problem I had with this was that it seemed to have a few dependencies and relied on an old version of d3, which had a big re-make recently. The second one is phylogeny.io which also offers an excellent web-app that you can drop a file into. Lastly, and this is the one that I have used on this website, is figtree.js. Figtree.js is not very closely related to actual figtree but is a javascript library for displaying trees on the web. I like the overall look and feel of the trees they produce! See my current implementation of a phylogeny of the British flora here.

Just to note here that the data for the phylogeny was contributed from people at Kew Gardens, the Royal Botanical Garden Edinburgh and the National Botanical Garden of Wales, and is to hopefully be published soon!
